## 📉 二、掉分尝试（已验证为负向 / 收益不明显）

| 类别 | 改动项 | 备注 / 失败原因猜想 |
|------|--------|---------------------|
| 🎨 **多模态特征** | 直接更换其余多模态特征 | 噪声大或对齐问题 |
|  | 多模态融合 | 融合方式可能不匹配主任务 |
| 🎯 **困难负样本** | 迭代困难负样本（当前 batch top10 难负 + 上 batch top50 难负） | 引入太多噪声 / 假负样本 |
| 🧠 **架构尝试** | 自回归训练 | 与召回任务目标不符 |
|  | HSTU 架构 | 结构与当前数据规模不匹配 |
|  | LIGR 架构 | 同上 |
|  | 增加更多 Norm | 过度正则化损害表达能力 |
| 📊 **损失函数** | BPR Loss | 不如 InfoNCE 充分利用全 batch 负样本 |
| 🔢 **量化/语义ID** | RQ-VAE，语义 ID | 训练不稳定或粒度问题 |
| 🎯 **辅助任务** | 引入点击位置辅助训练 | 无显著收益 |
|  | 曝光点击特征 | 特征穿越或分布偏移 |
| 🔀 **模型融合** | SWA 融合 | 相比单模 + 后处理收益不足 |
| 🎲 **采样策略** | 相似度采样 | 与 InfoNCE 配合不佳 |
